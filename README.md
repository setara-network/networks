# Setara Networks

Genesis files and network configurations for Setara blockchain networks.

## Networks

| Network | Chain ID | Status | Genesis |
|---------|----------|--------|---------|
| Testnet | `setara-testnet-1` | Active | [genesis.json](testnet/genesis.json) |
| Mainnet | `setara-1` | Coming Soon | — |

## Usage

```bash
# Download testnet genesis
curl -o ~/.setara/config/genesis.json \
  https://raw.githubusercontent.com/setara-network/networks/main/testnet/genesis.json
```
